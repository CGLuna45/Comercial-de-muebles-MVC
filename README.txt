Proyecto demo de catálogo Cedrika.

Archivos principales:
- index.php: inicio elegante con logo, categorías, misión, visión, reseñas y contacto.
- catalogo.php: búsqueda funcional y catálogo general.
- sala.php, comedor.php, escritorio.php: 8 productos por categoría con anclas para búsqueda exacta.
- carrito.php y pago-exitoso.php: páginas visuales de carrito y pago.

Ejecutar en XAMPP dentro de htdocs y abrir:
http://localhost/catalogo-cedrika/
